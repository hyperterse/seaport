# Changelog

## 0.1.0

- Added the initial Seaport crate scaffold.
- Added the initial `seaport` CLI command surface.
- Replaced the Rust example with a Seaport task directory example.
- Added GitHub Actions build/release workflows and an installer script.
- Added deterministic case ordering and deterministic run IDs.
- Added structured seaport errors with stable error codes.
- Added in-memory telemetry with stable sequence numbers and sorted attributes.
- Added unit tests, integration tests, an example, and a benchmark harness.
