## Caveat

This benchmark compares oracle harness overhead on the same task. Seaport uses
its local oracle runner. Harbor uses its Docker-backed oracle task execution.
This is a valid measurement for this benchmark path, but it is not yet a claim
that Seaport has feature parity with Harbor's full sandbox, registry, viewer, or
agent integration surface.
